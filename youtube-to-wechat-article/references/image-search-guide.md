# 网络图片搜索指南

## 搜索策略

### 优先使用网络图片的场景
1. **真实产品和品牌**：实际存在的产品、公司logo、品牌形象
2. **真实人物**：公司CEO、行业专家、名人照片
3. **数据图表**：已发布的统计图表、研究报告中的数据可视化
4. **实际场景**：办公室、工作场景、会议场景等真实照片
5. **技术界面**：软件截图、产品界面、应用演示

### 使用AI生成的场景
1. **抽象概念**：人机协作、未来愿景、创新理念等抽象主题
2. **定制图表**：需要特定数据和样式的流程图、对比图
3. **概念插画**：不存在的场景、想象中的未来、创意表达
4. **统一风格**：需要与品牌色彩和视觉风格完全一致的图片

## 搜索方法

### 使用search工具
```
search(
    type="image",
    queries=["关键词1", "关键词2", "关键词3"]
)
```

### 搜索关键词技巧
1. **具体化**：使用具体的产品名、品牌名、概念名
2. **多语言**：同时使用中文和英文关键词
3. **场景化**：描述具体的使用场景或应用环境
4. **风格化**：加上"professional"、"modern"、"business"等风格词

## 图片评估标准

### 质量评估
- **分辨率**：至少1920×1080px，适合微信公众号使用
- **清晰度**：图片清晰，没有模糊或像素化
- **构图**：主体明确，构图合理，视觉焦点清晰
- **色彩**：色彩自然，不过度饱和或失真

### 相关性评估
- **主题匹配**：图片内容与文章段落主题高度相关
- **信息准确**：图片中的信息（如数据、文字）准确无误
- **时效性**：图片反映当前或最新的状态，不过时
- **文化适配**：适合中文读者，没有文化冲突

### 版权评估
- **来源可信**：来自Unsplash、Pexels等免费图库
- **授权明确**：CC0、Public Domain或明确的商业使用授权
- **无水印**：图片上没有明显的水印或版权标识
- **可商用**：确认可用于商业性质的微信公众号文章

## 图片处理流程

### 下载和保存
1. 使用search工具搜索图片，获得下载路径
2. 图片自动下载到`/home/ubuntu/Downloads/`目录
3. 使用shell工具将图片复制到工作目录
4. 重命名图片为有意义的文件名

```bash
cp /home/ubuntu/Downloads/image_*.jpg /home/ubuntu/article_images/01_cover_image.jpg
```

### 图片优化
1. **尺寸调整**：使用shell命令调整图片尺寸
2. **格式转换**：统一转换为JPG格式
3. **压缩优化**：适当压缩以减小文件大小

```bash
# 使用ImageMagick调整尺寸和压缩
convert input.jpg -resize 1920x1080 -quality 85 output.jpg
```

## 混合策略实施

### 决策流程
```
对于每张需要的图片：
1. 明确图片主题和用途
2. 使用至少 2-3 个关键词变体（中文+英文）进行网络搜索
3. 评估搜索结果
   - 找到合适图片（质量达标 + 版权合规）→ 下载并使用
   - 未找到合适图片 → 尝试更换关键词再搜索一次
4. 穷尽搜索手段仍未找到 → 使用AI生成（最后手段）
5. 记录图片来源和版权信息
```

> ⚠️ **重要原则**：AI 生成是最后手段，不是默认选项。每张图片都必须先经过充分的网络搜索，再决定是否需要AI生成。

## 版权合规指南

### 免费图库推荐
- **Unsplash**：高质量免费图片，CC0授权
- **Pexels**：免费商用图片，无需署名
- **Pixabay**：免费图片和视频，简化授权
- **Freepik**：部分免费，需注意授权条款

### 避免版权问题
1. **不使用**：Getty Images、Shutterstock等付费图库的水印图
2. **不使用**：明显有版权标识的图片
3. **不使用**：来源不明的图片
4. **不使用**：名人肖像权保护的图片（未经授权）

### 版权声明
在文章末尾或图片说明中注明：
- "本文部分图片来自Unsplash/Pexels，遵循CC0协议"
- "本文部分图片由AI生成，无版权问题"

## 常见问题

### Q: 搜索不到合适的图片怎么办？
A: 尝试更换关键词，使用英文搜索，或调整搜索策略。如果仍然找不到，使用AI生成。

### Q: 如何判断图片是否可商用？
A: 优先使用Unsplash、Pexels等明确标注CC0或免费商用的图库。避免使用来源不明的图片。

### Q: 网络图片和AI生成图片风格不一致怎么办？
A: 在AI生成时，使用网络图片作为reference参数，保持视觉风格一致。

### Q: 需要多少张网络图片和AI生成图片？
A: **目标是尽可能多地使用网络图片**。除非确实无法找到合适的网络图片，否则不应使用AI生成。理想情况下，6-8张图片中网络图片占比应超过70%，甚至达到100%。AI生成仅用于填补网络搜索无法满足的空缺。
