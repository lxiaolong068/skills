# 微信公众号排版指南

## 字体规范

| 内容类型 | 字体大小 | 加粗 | 颜色 |
| :--- | :--- | :--- | :--- |
| 主标题 | 18-20px | 是 | #333333 或 #0066CC |
| 副标题/引言 | 14-16px | 否 | #999999（斜体） |
| 一级标题 | 16-18px | 是 | #0066CC |
| 二级标题 | 14-16px | 是 | #333333 |
| 正文 | 14-16px | 否 | #333333 |
| 强调词汇 | 14-16px | 是 | #FF6B6B 或 #0066CC |
| 辅助信息 | 12-14px | 否 | #999999 |

## 间距规范

### 段落间距
- 正文段落：1.5倍行距
- 段落之间：空一行
- 章节之间：空一行 + 分隔线

### 列表间距
- 列表项之间：0.5倍行距
- 列表与正文之间：1倍行距

## 图片规范

### 尺寸建议
- 横版图片：900-1080px宽度
- 竖版图片：600-800px宽度
- 封面图：1080×600px（推荐）

### 图片位置
1. **封面图**：标题下方，第一段前
2. **章节配图**：章节标题后，正文前
3. **说明配图**：相关段落后
4. **结尾配图**：最后一段后

### 图片说明
- 位置：图片下方
- 字体：12-14px，灰色 (#999999)
- 格式：简短描述，可选

## 表格规范

### 表格样式
- 表头背景色：#E8F4F8（浅蓝色）
- 表格边框：1px 灰色 (#CCCCCC)
- 行高：1.8倍
- 字体大小：14px

### 表格使用
- 对比信息：使用表格
- 数据展示：使用表格
- 移动端友好：列数不超过3列

## 配色方案

### 主色调
- 品牌色：#0066CC（深蓝色）- 标题、重点词汇
- 辅助色：#FF6B6B（红色）- 警示、重要数据
- 背景色：#F5F5F5（浅灰色）- 代码块或引用块

### 文字颜色
- 标题：#0066CC 或 #333333
- 正文：#333333（深灰色）
- 强调：#FF6B6B（红色）或 #0066CC（蓝色）
- 辅助信息：#999999（浅灰色）

## 特殊格式

### 引用块
```
背景色：#E8F4F8（浅蓝色）
文字色：#0066CC
边框：左边 3px #0066CC
内边距：10-15px
```

### 重点提示块
```
背景色：#FFF3CD（浅黄色）
文字色：#333333
边框：左边 3px #FF6B6B
内边距：10-15px
```

### 代码块
```
背景色：#F5F5F5（浅灰色）
文字色：#333333
字体：等宽字体
边框：1px #CCCCCC
内边距：10px
```

## 发布建议

### 发布时间
- **工作日**：周二至周四的上午10-11点
- **周末**：周六周日的下午2-3点
- **避免**：周一早上（竞争激烈）、晚上11点后

### 推送文案
- 标题：与文章标题一致或简化版
- 摘要：100-150字，提炼核心观点
- 配图：使用封面图

### 互动引导
- 文末提问：引导读者留言
- "在看"引导：提醒读者点赞
- 相关推荐：推荐历史文章

## 编辑工具推荐

### 微信官方编辑器
- 优点：官方支持，稳定可靠
- 缺点：功能相对简单

### 第三方编辑器
1. **秀米**：模板丰富，样式多样
2. **135编辑器**：专业功能，排版效果好
3. **新榜编辑器**：功能全面，数据分析

## 发布前检查清单

- [ ] 标题吸引人，字数在15-25字之间
- [ ] 首段开篇引人入胜，100-150字
- [ ] 所有小标题清晰，层级分明
- [ ] 图片位置合理，尺寸统一
- [ ] 图片已压缩（每张<500KB）
- [ ] 表格格式清晰，数据准确
- [ ] 强调词汇突出，颜色搭配协调
- [ ] 段落间距合理，行距1.5倍
- [ ] 没有错别字或语病
- [ ] 链接正确，没有失效链接
- [ ] 整体阅读时间在5-8分钟之间
- [ ] 在手机端预览效果满意

## 移动端优化

### 文字大小
- 正文不小于14px
- 标题不小于16px
- 确保在小屏幕上清晰可读

### 图片优化
- 宽度不超过1080px
- 压缩文件大小（<500KB）
- 避免过多细节（移动端难以辨识）

### 交互优化
- 按钮和链接足够大（易于点击）
- 避免过长的横向滚动
- 表格在移动端可读（列数限制）
