# 微信公众号写作风格指南

## 标题写作技巧

### 标题长度
- 理想长度：15-25个字
- 避免过长或过短

### 标题类型
1. **疑问式**：引发好奇，如"那个能替你打工的AI来了，营销人的饭碗要被端走了？"
2. **数字式**：清晰具体，如"3个方法让你的效率提升10倍"
3. **对比式**：制造反差，如"从月薪5千到年薪50万，他只做对了一件事"
4. **悬念式**：吊足胃口，如"这个功能，99%的人都不知道"

### 标题禁忌
- 避免标题党和夸大其词
- 避免使用过多符号和emoji
- 避免政治敏感词汇

## 开篇引言技巧

### 引言长度
- 100-150字为宜
- 快速进入主题

### 引言方法
1. **名言引用**：借助权威观点，如"'AI不会取代你，但会用AI的人会。'"
2. **场景描述**：带入情境，如"当别人还在讨论AI能不能写出'金句'文案时..."
3. **数据冲击**：用数据震撼，如"硅谷一夜蒸发2万亿"
4. **问题引入**：提出痛点，如"你是否也有这样的焦虑？"

## 段落结构建议

### 段落长度
- 每段3-5句话
- 避免超过8行的长段落
- 移动端阅读体验优先

### 段落间距
- 段落之间空一行
- 使用分隔线区分大章节

### 层级结构
```
# 一级标题（章节）
## 二级标题（子章节）
### 三级标题（要点）

正文段落...

**强调内容**
```

## 数据呈现方式

### 表格使用
- 对比信息使用表格
- 表头清晰，数据准确
- 移动端友好的列数（建议不超过3列）

### 数据强调
- 使用**加粗**突出关键数据
- 使用颜色区分正负变化
- 使用箭头表示趋势（↑↓）

### 案例引用
- 具体案例 + 数据支撑
- 来源可信，避免虚构
- 适当使用引用格式

## 结尾写作技巧

### 结尾类型
1. **总结式**：回顾要点，强化记忆
2. **励志式**：鼓舞人心，传递正能量
3. **互动式**：提出问题，引导留言
4. **行动式**：给出具体建议，促进转化

### 结尾长度
- 100-200字为宜
- 避免拖沓冗长

## 避免机器写作痕迹

### 语言风格
- 使用自然的口语化表达
- 避免过于正式的书面语
- 适当使用网络流行语（但不过度）

### 句式变化
- 长短句结合
- 避免重复的句式结构
- 使用反问、设问增加互动感

### 情感投入
- 表达真实的观点和态度
- 使用第一人称增加亲切感
- 适当表达情绪（惊讶、担忧、兴奋等）

### 避免的表述
- "首先...其次...最后..."（过于机械）
- "综上所述"（过于正式）
- "值得一提的是"（过于套路）
- 过多的并列结构

### 推荐的表述
- "说到这里，你可能会想..."
- "别急，让我们..."
- "有意思的是..."
- "更重要的是..."

## 微信公众号特有要素

### 排版美化
- 使用秀米、135编辑器等工具
- 统一的配色方案
- 合理的图文搭配

### 互动引导
- 文末提问引导留言
- "在看"和"转发"引导
- 相关文章推荐

### SEO优化
- 标题包含关键词
- 文章标签设置
- 摘要精心撰写

## 内容质量标准

### 信息密度
- 避免空洞的废话
- 每段都有实质内容
- 数据和案例充实

### 可读性
- 逻辑清晰，层次分明
- 过渡自然，衔接流畅
- 专业术语适当解释

### 原创性
- 避免抄袭和洗稿
- 有独特的观点和见解
- 融入个人经验和思考
